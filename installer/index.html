Install TwinTrack

Put a live departure board onto a new Spotpear ESP32-C3 Mini TV directly from your browser.

Check the hardware

Only continue with the Spotpear ESP32-C3 1.44-inch Mini TV: ESP32-C3, 4 MB flash, 128 × 128 ST7735 display.

Firmware for another ESP32 board may install successfully but leave its display and buttons unusable until the correct image is restored.

Choose the display edition

Both editions provide the same departures, first-boot Wi-Fi setup, fallback access point, buttons, station web UI, and twintrack.local address.

Install in four steps

  1. Use Chrome or Edge on a Windows, macOS, or Linux computer.
  2. Connect the device directly with a USB data cable and close any serial monitor.
  3. Choose an edition, select the ESP32-C3 serial device, and confirm installation.
  4. Keep the cable connected until installation reaches 100% and TwinTrack restarts.

Existing device? The installer can preserve its saved Wi-Fi and train settings. A full-device erase returns it to first-boot setup.

Complete first-boot setup

  1. Read the temporary network name and password shown on the TwinTrack display.
  2. Join the displayed TwinTrack-XXXX network.
  3. Open the captive portal or browse to 192.168.4.1.
  4. Save the home Wi-Fi details, then open twintrack.local to choose stations.

Manual recovery

Each download is a merged ESP32-C3 image for offset 0x0. Use these if browser installation is unavailable.
