Install TwinTrack

Put a live departure board onto a supported ESP32 display directly from your browser.

Check the hardware

Choose the exact hardware below: either the Spotpear ESP32-C3 1.44-inch Mini TV, or an ESP32-WROOM-32D wired to a black/white/red WeAct 2.9-inch 296 × 128 SSD1680 e-paper module.

Firmware for the wrong target may install successfully but leave its display unusable until the correct image is restored.

Choose the hardware and display edition

All editions provide the same departures, first-boot Wi-Fi setup, fallback access point, and station web UI. The Mini TV uses twintrack.local; the e-paper build uses twintrack-paper.local.

Install in four steps

  1. Use Chrome or Edge on a Windows, macOS, or Linux computer.
  2. Connect the device directly with a USB data cable and close any serial monitor.
  3. Choose the matching hardware edition, select its serial device, and confirm installation.
  4. Keep the cable connected until installation reaches 100% and TwinTrack restarts.

Existing device? The installer can preserve its saved Wi-Fi and train settings. A full-device erase returns it to first-boot setup.

Complete first-boot setup

  1. Read the temporary network name and password shown on the TwinTrack display.
  2. Join the displayed TwinTrack-XXXX network.
  3. Open the captive portal or browse to 192.168.4.1.
  4. Save the home Wi-Fi details, then open the .local address shown on the display to choose stations.

Manual recovery

Each download is a merged image for offset 0x0. Use these if browser installation is unavailable.
